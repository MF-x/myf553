# myf553
Digital Image Processing Experiment

数字图像处理实验

---

## Features

1. 显示BMP图像

2. 状态栏显示坐标和图片RGB值

3. 24位真彩图像灰度化

4. 重载图像

5. 显示8位灰度图像的灰度直方图

6. 实现点运算

7. 实现均衡化

8. 实现标准傅立叶变换及其反变换

9. 实现快速傅立叶变化及其反变化

10. 实现均值滤波

11. 实现中值滤波

12. 实现梯度锐化

13. 实现拉普拉斯锐化

14. 实现低通滤波

15. 实现高通滤波
